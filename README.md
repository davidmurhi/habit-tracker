# Habit Tracker

A command-line habit tracking application built in Python.

## Features

- Add habits to track
- View all habits with current streak counts
- Mark habits as done (increments streak)
- Delete habits
- Persistent storage — habits are saved between sessions using JSON
- Input validation and error handling

## Tech Stack

- Python 3
- JSON for data persistence
- pytest for automated testing

## How to Run

1. Clone the repository
2. Run the app: `python3 main.py`
3. Run the tests: `pytest test_habits.py -v`

## What I Learned

The biggest challenge was syntax — trying to remember exact commands 
under pressure. What actually helped was shifting my mindset: I don't 
need to memorise everything, I just need to understand what I want the 
program to do next and work from there. Loops were the concept I 
struggled with most, but by the end I'd used them enough across 
different problems that they started to feel natural. If I built this 
again, I'd be more deliberate from the start — planning the structure 
before writing, and committing smaller, more frequent changes rather 
than rewriting large sections at once.
